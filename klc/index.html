Type-A Kazhdan-Lusztig Calculator

Compute KL polynomials for Sn (n ≤ 16).

0-indexed continuous notation (e.g., 0123)
0-indexed continuous notation (e.g., 3210)

Result Px,w(q)

Coefficients are listed starting from degree 0. For example, 1,3,2 represents 1 + 3q + 2q².

· Based on Gregory S. Warrington's KLC

Documentation

Permutation Notation

Permutations are represented in 0-indexed one-line notation over {0, 1, ..., n-1}.

  • Valid (S4): 0213
  • For elements 10 and above, use lowercase letters: a=10, b=11...
  • Example (S12 identity): 0123456789ab

Mathematical Constraints

Px,w(q) is computed only when x ≤ w in the Bruhat order. If x ¬≤ w, the program returns 0.

Result Format

The output is a comma-separated list of coefficients starting from degree 0. For example, 1,3,2 represents 1 + 3q + 2q².
